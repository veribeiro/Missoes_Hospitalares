# USO_IA
* Qual ferramenta de IA foi usada? <br>
Foi utilizado o Gemini.

* Para quais partes do projeto a IA foi usada? <br>
Foi utilizado na parte da criação de envio de API em JSON simulado, organizar prioridades em ordenação e me orientou como criar elementos HTML em JavaScript.

* Quais trechos foram revisados, modificados ou corrigidos pelo candidato? <br>
Os trechos que foram revisados foram alerta() para que todas as condições sejam 'if', que exiba o id de qual missão que falhou e que apareça mais de um alerta na tela, caso for necessário. Um outro trecho que foi revisado foi a função enviarMissaoHospitalar() para garantir que todos os dados fossem utilizados na parte do cadastro.

* Quais problemas a IA gerou? <br>
O principal problema que a IA gerou foram pequenos erros de integração em JSON, que logo foram resolvidos.

* O que o candidato aprendeu ou precisou corrigir? <br>
Aprendi como armazenar dados em formato JSON e como integrar isso em HTML.

* Quais partes do código o candidato considera mais importantes para a avaliação? <br>
As principais partes que considero importantes são a visualização de todas as missões na lista, a telemetria do robô e os alertas.
